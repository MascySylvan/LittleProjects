# Product Overview

LittleProjects is a collection of standalone Java desktop applications focused on visual simulations and games. Each sub-project is an independent program with its own `main` method.

## Sub-Projects

- **Snake (manual)** — Classic snake game with keyboard controls, grid-based movement, and collision detection.
- **Snake (AI)** — AI-driven snake that uses a Hamiltonian cycle with shortcutting to fill the grid. Includes a menu UI with configurable grid sizes and manual/AI toggle.
- **Self-Avoiding Walk** — Visual simulation of a random self-avoiding walk on a 2D grid with backtracking.
- **Cube Timer** — Stopwatch utility for timing Rubik's cube solves, toggled with spacebar.
